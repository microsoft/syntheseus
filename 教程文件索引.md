## 🤔 遇到问题？

### 看不到文件？
```bash
# 刷新文件浏览器或运行
ls -la /workspace/ | grep 教程
```

### 无法运行Python示例？
```bash
# 检查Python环境
python --version
# 检查syntheseus安装
python -c "import syntheseus; print(syntheseus.__version__)"
```

### Jupyter无法打开？
```bash
# 安装Jupyter
pip install jupyter
# 或使用JupyterLab
pip install jupyterlab
jupyter lab
```
